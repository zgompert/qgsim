# qgsim

# Installation

You can install this package within R directly from GitHub.

```R
## install and load devtools
install.packages("devtools")
library(devtools)
## install qgwim
devtools::install_github("zgompert/qgsim")
## load the qgsim package
library(qgsim)
```
